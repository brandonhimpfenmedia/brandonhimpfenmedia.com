# Brandon Himpfen Media (Jekyll)

A platform-first studio homepage + blog designed to run on GitHub Pages.

## Local development

1. Install Ruby + Bundler
2. Run:
   - `bundle install`
   - `bundle exec jekyll serve`
3. Open http://localhost:4000

## Deploy on GitHub Pages

1. Push this repo to GitHub
2. Go to **Settings â†’ Pages**
3. Choose:
   - Source: **Deploy from a branch**
   - Branch: `main` (or `master`)
   - Folder: `/ (root)`

## Digital properties

Update `_data/properties.yml` to change the cards on the homepage.
